A modern, interactive design agency homepage built with Next.js 14, featuring stunning animations, dark/light mode, and an engaging user experience.

![Hero Section](https://images.unsplash.com/photo-1551288049-bebda4e38f71?w=1200)

## ✨ Features

### 🎯 **Core Features**
- **Responsive Design** - Perfectly optimized for mobile, tablet, and desktop
- **Dark/Light Mode** - Seamless theme switching with localStorage persistence
- **Interactive UI** - Engaging animations and hover effects throughout
- **Smooth Scrolling** - Implemented with Lenis for butter-smooth navigation

### 🎨 **Interactive Elements**
- **Animated Orbit Design** - Rotating rings with floating icons and particles
- **Background Dots** - Dynamic particle system with independent animations
- **Welcome Message** - Friendly greeting for first-time visitors
- **Floating Icons** - Emoji icons with smooth floating animations
- **Orbiting Dots** - Pulsing dots that circle the main logo

### 📱 **Sections**
- **Hero Section** - Eye-catching introduction with interactive orbit design
- **Services Section** - 4 service cards with hover glow effects
- **Portfolio Section** - Masonry grid layout with project overlays
- **Contact Section** - Functional contact form with EmailJS integration
